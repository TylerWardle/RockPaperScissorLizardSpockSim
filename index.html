    Welcome to the landing page for my Rock Paper Scissors Lizard Spock Tournament Application! This application simulates a single elimination tournament of RPSLS.

    Basically, you can add, remove or update players in the starting bracket. When you are ready just press play for some RPSLS with live and simulated players. This application is powered by Swing and backed by a H2 DB. If you wish to see the data persisted to H2, open up the local console after the application has started.

    URL: http://localhost:8082 (H2 will start a server for you) and login with the following settings:

    • Generic H2 (Embedded)
    • org.h2.Driver
    • jdbc:h2:mem:testdb
    • blank username and password

    Click here to download my application. After you unzip this application into a suitable folder, "cd" into that directory and run: java -jar rockpaperscissorslizardspocksim-0.0.1-SNAPSHOT.jar
